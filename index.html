Drinkups is an app for developers who like Github drinkups, built by a developer who goes to Github drinkups. It's not a cheesy, social networking layer on top of them — it's simply a utility for reminding and directing you to them.

If you need to check-in on Foursquare, tweet out the link to Github's blog post, or call the bar — all that functionality is built in, too. But, mostly it's just an app that you can open, interact with, and close — all for free, without ads or external noise.

Think of it as the information you wanted to know about the next Github drinkup location, in one simplified interface.

Made by @zachwill.
